Open Network Ports

Solstice uses all TCP/IP standard network traffic to communicate across all the required and optional components of the Solstice system. The following may be required for Solstice to work correctly:

  • Required network ports/routes must be open.
  • The network Solstice is deployed must allow peer-to-peer TCP connections.

  • Access to a few external sites must be allowed for pod activation, licensing, updates, and Solstice Cloud Management.
